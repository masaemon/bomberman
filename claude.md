# Bomberman Game - 開発ドキュメント

## プロジェクト概要
ブラウザベースの2Dボンバーマンゲーム（シングルプレイヤー、AI対戦）

## 技術スタック
- **言語**: TypeScript
- **ビルドツール**: Vite
- **ゲームエンジン**: Phaser 3
- **プラットフォーム**: Webブラウザ

## ゲーム仕様

### 基本設定
- **フィールドサイズ**: 設定可能（デフォルト: 横30 × 縦15マス）
- **プレイヤー**: 1人 vs AI 3体
- **破壊可能な壁**: フィールドの約40%
- **タイルサイズ**: 32px × 32px（予定）

### ゲームルール

#### パワーアップアイテム
1. 爆弾数増加（同時設置可能数）
2. 爆風範囲拡大
3. 移動速度アップ

#### フィールド縮小機能
- **開始時間**: ゲーム開始から60秒後
- **縮小ペース**: 15秒ごとに外周1マスずつ縮小
- **縮小エリア**: プレイヤー・AIがいる場合はダメージ

#### AI仕様
- **初期実装**: シンプルなランダム移動 + 爆弾設置
- **将来**: より戦略的な行動を追加可能

### 勝利/敗北条件
- **勝利**: すべてのAIを倒す
- **敗北**: プレイヤーが爆風またはフィールド縮小で死亡

```
│   │   └── MenuScene.ts        # タイトル/設定画面（将来）
│   ├── entities/
│   │   ├── Player.ts           # プレイヤーキャラクター
│   │   ├── Enemy.ts            # 敵AI
│   │   ├── Bomb.ts             # 爆弾
│   │   ├── Explosion.ts        # 爆発エフェクト
│   │   └── PowerUp.ts          # パワーアップアイテム
│   ├── map/
│   │   ├── Map.ts              # マップ生成・管理
│   │   └── Tile.ts             # タイル情報
│   ├── ai/
│   │   └── SimpleAI.ts         # AI行動ロジック
│   └── utils/
│       └── FieldShrink.ts      # フィールド縮小ロジック
└── assets/                     # 画像・音声（将来）
```
